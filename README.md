# 🐱 全网页互动猫咪浮窗

一个可爱的 Chrome 浏览器扩展，为你的网页添加一只可交互的猫咪浮窗。猫咪会自动做出各种萌萌的动作，还能响应你的互动！

---

## ✨ 功能特性

- 🎯 **自由拖拽**：鼠标拖动猫咪到屏幕任意位置，智能边界检测
- 💖 **互动反馈**：点击猫咪触发歪头、脸红、闭眼、飘爱心等可爱动画
- 🎨 **自动动画**：自动晃耳朵、摇尾巴、随机眨眼，让猫咪栩栩如生
- 🌐 **全网页支持**：兼容所有 http/https 网页
- ⚡ **轻量级**：纯 CSS + JavaScript 实现，不影响网页性能

---

## 🚀 安装步骤

### 第一步：准备扩展文件

1. **创建扩展文件夹**
   
   新建文件夹命名为 `cat_float_extension`，在该目录下创建 `icons` 子目录用于存放浏览器扩展图标。

2. **创建核心文件**
   
   在扩展文件夹中创建以下 3 个核心文件：
   - `manifest.json` - 扩展的核心配置文件（必须）
   - `content.css` - 猫咪的样式代码
   - `content.js` - 猫咪的交互逻辑代码

### 第二步：在浏览器中加载扩展

**Chrome 浏览器加载步骤：**

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车，进入扩展管理页面
2. 打开页面右上角的 **开发者模式** 开关（开启后会从灰色变为蓝色）
3. 点击页面左上角的 **"加载已解压的扩展程序"** 按钮
4. 在弹出的文件选择窗口中，选择刚才创建的 `cat_float_extension` 文件夹
   > ⚠️ **注意**：选择整个文件夹，而不是文件夹内的单个文件
5. 点击 **"选择文件夹"** 完成加载
6. 加载成功后，扩展列表中会出现 **"全网页互动猫咪浮窗"**

## 🎮 使用说明

### 启动猫咪

安装完成后，打开任意网页（如百度、知乎、B 站等），页面右下角会自动出现猫咪浮窗。

> 📌 **适用范围**：仅支持 `http://` 或 `https://` 开头的网页

### 互动功能

| 操作 | 效果 |
|------|------|
| 🖱️ **拖动猫咪** | 按住猫咪可拖动到屏幕任意位置，智能防止拖出屏幕外 |
| 👆 **点击猫咪** | 触发歪头、脸红、闭眼、飘爱心等互动动画 |
| 👀 **观察猫咪** | 猫咪会自动晃耳朵、摇尾巴、随机眨眼 |

### 启停控制

- **暂时禁用**：进入 `chrome://extensions/` 页面，关闭该扩展的开关
- **重新启用**：在扩展管理页面重新打开开关即可

---

## 💡 提示

- 首次安装后，可能需要刷新已打开的网页才能看到猫咪
- 扩展仅在符合条件的网页上生效（http/https 协议）
- 关闭扩展后，需要刷新页面才能彻底移除猫咪浮窗
